# Playhead Operations

## Quick Start (Makefile)

We use a `Makefile` to simplify development commands.

### Verified Commands

- **Start All (Frontend + Backend)**:
  ```bash
  make dev
  ```
  *Starts `npm run dev` and `uvicorn` in parallel.*

- **Frontend Only**:
  ```bash
  make dev-web
  ```

- **Backend Only**:
  ```bash
  make dev-backend
  ```

- **Install Dependencies**:
  ```bash
  make install
  ```

- **Clean Caches**:
  ```bash
  make clean
  ```
